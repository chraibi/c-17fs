# Linux

## g++ >= 8
- `cd build`
- `cmake  -DCMAKE_CXX_COMPILER=g++-8  ..` 
- `make`

## clang >=8

- `cd build`
- `cmake  -DCMAKE_CXX_COMPILER=clang++-8  ..` 
- `make`
